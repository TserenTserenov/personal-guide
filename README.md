# Персональное руководство

Это твоё персональное руководство по развитию в IWE. Репозиторий - контейнер, куда система (Портной) складывает занятия, а ты их проходишь и закрываешь. На твоих результатах система учится и делает следующие занятия точнее.

Этот файл (README) описывает сам репозиторий и не трогается автоматикой - он для тебя и для того, кто откроет репозиторий впервые.

## Три уровня времени

- **Месяц (корень):** медленный тренд - когда что менялось (квалификация, ступень). Дописывается сверху, чтобы пролистать всю историю.
- **Неделя (`lesson/<год>-W<номер>.md`):** всё твоё состояние недели в одном файле - ступень, узкое место, методы, тема и задание недели, ритм.
- **День (`lesson/<дата>.md` + `panel/<дата>.md`):** занятие дня (задание + твоя работа) и снимок табло за день.

## Как это работает за один проход

1. Утром система собирает занятие дня и снимок табло, кладёт в `lesson/` и `panel/`.
2. В начале недели собирает недельный файл со всем состоянием недели.
3. Ты открываешь занятие (через IWE командой `/lesson` или просто читаешь файл), выполняешь и дописываешь результат в тот же файл дня.
4. Закрываешь занятие (`/lesson-close`) - проставляется отметка и время, результат уходит в систему и обновляет твой цифровой двойник.
5. Когда меняется крупное (получил квалификацию, перешёл на ступень) - система дописывает строку сверху в тренд `milestones.md`.
6. В конце недели весь недельный пакет (`lesson/` и `panel/`) уезжает в архив `history/`, а в `lesson/` появляется новая неделя.

## Структура репозитория

| Путь | Что это | Кто пишет |
|------|---------|-----------|
| `README.md` | этот файл, про репозиторий | вручную |
| `guide.json` | служебный индекс для веб-страницы | система |
| `milestones.md` | тренд развития: когда что менялось (квалификация, ступень), новое сверху | система дописывает сверху; ты тоже можешь |
| `lesson/` | пакет текущей недели (недельный файл + дни) | система + ты (низ файла дня) |
| `panel/` | дневные снимки табло | система |
| `history/` | архив по годам и месяцам | система + ты (рефлексии) |

### Пакет текущей недели - `lesson/`

```
lesson/
  2026-W25.md      ← недельный файл: ступень, узкое место, методы, тема, задание, ритм
  2026-06-16.md    ← занятие дня: задание сверху + твоя работа снизу (до 7 дней)
  ...
```

Всё, что касается недели (ступень, узкое место, методы, мировоззренческая линза), пишется в **недельный файл** - отдельных profile/worldview/methods больше нет.

### Дневные снимки табло - `panel/`

```
panel/
  2026-06-16.md    ← показатели дня (множитель, поток, баллы, ступень)
```

Данные табло считаются и хранятся в системе (Neon); в репозиторий приходит только готовый дневной снимок-показ. Репозиторий - место показа, не место сбора.

### Тренд развития - `milestones.md` (корень, месячный ритм)

Дописывается сверху, чтобы пролистать всю историю и увидеть, когда что менялось:

```
## 2026-06-17 - Ступень 5, квалификация Реформатор
## 2026-03-10 - Переход на ступень 4
## 2026-01-01 - Получил квалификацию X
```

### Архив - `history/`

```
history/
  2026/
    06/
      2026-W25.md            ← заархивированная неделя
      2026-06-16.md          ← занятие дня (задание + твоя работа)
      2026-06-16-panel.md    ← табло того же дня (рядом по дате)
      reflection-2026-06-18.md
```

В конце недели весь пакет недели уезжает в `history/<год>/<месяц>/`, где файлы сортируются по дате и лежат рядом.

## Что трогать руками, а что нет

- **Не трогай:** `guide.json`, недельный файл, **верх файла дня (задание)**. Пересобирается автоматически.
- **Твоё:** **низ файла дня** (что сделал, заметки, рефлексия), закрытие занятия (`/lesson-close`), свои вехи в `milestones.md`, рефлексии в `history/`.
- Хочешь поправить само задание - не правь файл, а дай обратную связь: она влияет на следующую сборку.

## Где смотреть руководство - три способа

1. **Прямо в гите / VS Code** - открываешь файлы `lesson/` как есть.
2. **Через IWE** - команда `/lesson` открывает занятие дня и ведёт по нему.
3. **На сайте** - если подключить репозиторий к guide.system-school.ru (приложение «Aisystant Personal Guide»), руководство видно как веб-страница.

## Тайный или явный репозиторий

- **Тайный (приватный):** руководство видишь только ты. Подходит, если в нём личные данные. Веб-страница работает через приложение с доступом.
- **Явный (публичный):** руководство можно показать другим по ссылке на гит.

Выбор твой, система работает в обоих режимах. Личные данные всегда остаются у тебя; на сайт уходит только то, что ты подключил.

## Как лучше работать - коротко

- Утром: `/lesson` - открыть занятие.
- Сделал задание (45-90 мин), дописал результат в тот же файл - `/lesson-close`.
- Раз в неделю смотри недельный файл; раз в месяц пролистай `milestones.md`.
- Не редактируй сгенерированное; влияй на него обратной связью.
