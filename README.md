## Как читать

Начни с **daily/2026-05-15.md** — там ровно два абзаца, что делать сегодня. Потом обсуди с Claude Code (если нужен чат), закрой слот. Завтра откроешь **daily/2026-05-16.md**, и так неделю.

**Главное:** план на неделю облегчённый. Это восстановительная неделя. Не добавляй методы, не ставь цели вроде «выпустить фичу». Одна помидорка в день, фокус на том, чтобы поток вернулся.
