# Методы: bottleneck W (мировоззрение/окружение)

> Ступень 3 (Систематический). Bottleneck — W: мировоззрение пока на 3, тогда как M1=4, M2=5, M4=4.
> На ступени 3 в работе все 8 методов минимум в степени 1. Один метод идёт «вглубь» (до степени 3), остальные удерживаются на 1–2.

## Фокусные методы под bottleneck W

**Bottleneck W близок к M4 (мировоззрение)** — методы те же:

### 1. METHOD.007 — формирование окружения (до степени 2)

**Первая итерация — обязательна на ступени 3, даже если bottleneck другой.**

Шаги:
1. Заполнить таблицу «Моё окружение»: команда, каналы (Telegram, Slack, RSS), места работы, информационные потоки.
2. Пометить каждую часть: оставить / исключить / добавить.
3. Выбрать **один кандидат** на изменение на текущую неделю.

| Категория среды | Примеры (Knowledge Worker) |
|-----------------|---------------------------|
| Команда | Андрей, Паша, Дима, Ильшат, Олег |
| Каналы информации | Telegram-каналы, GitHub subscriptions, RSS |
| Профессиональные сообщества | сообщество IWE, LLM/AI-конференции |
| Рабочие пространства | IWE/VS Code + Claude Code среда |

### 2. METHOD.008 — стратегирование (до степени 2)

Еженедельная сессия (воскресенье/понедельник):
- 6 стадий (по CAT.003)
- Список проектов с бюджетом и сроком
- Личная траектория: куда растёт IWE как система, куда растёт основатель

Вопрос на каждой сессии: «Какой C_act+1 меня ждёт через квартал?»

### 3. METHOD.005 — мышление проговариванием (в сообществе)

Инструмент входа в «окружение влияет»:
- Архитектурные обсуждения в команде
- Посты в клуб / профессиональное сообщество (1 заготовка/нед)
- Выделение понятий при разборе терминологии домена пользователей IWE

## Правило движения

- Не пытайся поднять часы до 10 — это маркер ступени 4. Ступень 3 про **стабильность 6 часов как норма**.
- Одна итерация таблицы «Моё окружение» = входная точка разворота наружу. Всегда делается, независимо от bottleneck.
- Попытка поднять все методы сразу → выгорание. Это документированный failure-mode ступени.
