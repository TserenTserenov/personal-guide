# Профиль: Реформатор (квалификация L7)

## Профиль квалификации

- **Степень:** Реформатор (PD.QUAL.001, L7 по европейской рамке EQF). Автономия уровня — **трансформация контекстов**: твои решения меняют правила игры для целых направлений и сообществ, а не только для собственных проектов. Выше только Общественный деятель (L8).
- **Что подтверждает уровень:** созданные системы и подходы, которыми пользуются другие; влияние на сообщество. Подтверждение из учётной системы — выступление на конференции 2025.
- **Учебная ступень (вторичный сигнал):** Проактивный (5). Учебный ритм устойчив (7 дней в неделю активности), провала по роли Ученика нет — мягкая рекомендация из Личного развития сейчас не нужна.

## RCS-снимок (вторичный, для сшивки с ЛР)

- Системность языка (W): 3
- Ресурс и собранность (M1): 5
- Мировоззрение в действии (M4): 4
- Методы саморазвития (M2): данных нет (по умолчанию 2 — не измеренный провал, рекомендацию не даём)

Из измеренных слотов ни один не просел ниже 3 — по учебной оси всё в норме, фокус целиком на рычаге Реформатора.

## Рычаг уровня

Главный рычаг — **выбор контекста для трансформации**: один правильно выбранный контекст важнее десятка улучшений внутри существующих правил. Метрика квартала — сдвиг контекста: появилась ли практика/подход, которой раньше не было и которую переняли другие. Тест — можешь назвать конкретных людей или команды, которые работают иначе из-за тебя.
